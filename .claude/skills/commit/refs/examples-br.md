# Exemplos em Português

## Simples

```
feat(auth): adicionar funcionalidade de redefinir senha
fix(api): resolver ponteiro nulo no serviço de usuário
refactor(hooks): simplificar lógica de gerenciamento de estado
```

## Com corpo

```
feat(forms): refatorar hooks para usar tipos gerados

- Adiciona parâmetro transformData ao useFormMutation
- Atualiza hooks de auth com mensagens de validação
- Adiciona hook useCreateSiteAuto para criação automática
```
